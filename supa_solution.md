# Problem

When you have a multi expense report, you can scroll in the preview to see all the expenses, but then no matter where you click, you are taken to the report page. This is unexpected as users would think it takes you to the expense you selected

# Solution

Change it so that:
- Clicking on the expense takes you to the expense itself
- If on mobile, it would push 2 pages: the report one and the expense one, so going back would take you to the report
- In order to make it easier to navigate to the report itself, add a grey `view` button to the report actions
