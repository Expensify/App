# XERO01 Error: Unable to Complete Action to Xero

If you see the error message **“XERO01 Error: Unable to complete action to Xero. Please reach out to Concierge for additional assistance.”**, it means there is a connection issue between Expensify and Xero.

This error prevents the report or transaction from syncing successfully to Xero.

---

## Why the XERO01 Error Happens

The XERO01 error indicates a disruption in communication between Expensify and Xero.

Common causes include:

- An expired or disconnected Xero connection  
- Authentication issues  
- Changes to Xero organization permissions  
- Temporary service interruptions  

Because this error relates to the integration itself, it cannot be resolved directly from the report.

---

## How to Fix the XERO01 Error

Please reach out to **Concierge** in Expensify.

On web:

1. Go to the **navigation tabs on the left**.
2. Select **Concierge**.
3. Start a new message and reference the **XERO01 Error**.

On mobile:

1. Tap the **navigation tabs on the bottom**.
2. Select **Concierge**.
3. Start a new message and reference the **XERO01 Error**.

Concierge will review your Xero connection and help restore the integration so you can complete the export.

---

# FAQ

## Should I retry the export before contacting Concierge?

You can try exporting again in case the issue was temporary. If the error persists, contact Concierge.

## Does the XERO01 Error affect all exports?

It can. If the Xero connection is disrupted, multiple reports or exports may fail until the connection is restored.
