# DESK52 Sync Error in QuickBooks Desktop Integration

If you see the error:

DESK52 Sync Error: Unable to set user as “preferred exporter” of the Workspace.

This means there are multiple QuickBooks Desktop connections configured within the same Workspace.

QuickBooks Desktop integrations require a single designated connection and exporter per Workspace.

---

## Why the DESK52 Sync Error Happens in QuickBooks Desktop

The DESK52 error typically occurs when:

- More than one QuickBooks Desktop connection exists in the Workspace.
- Multiple users have attempted to connect QuickBooks Desktop.
- The system cannot determine which connection should be the preferred exporter.

Only one QuickBooks Desktop connection can be active as the preferred exporter.

This is a configuration conflict, not a data issue.

---

# How to Fix the DESK52 Sync Error

This issue requires assistance from **Concierge**.

Please contact Concierge and include:

- The Workspace name.
- Confirmation that multiple QuickBooks Desktop connections may exist.
- The approximate time the error occurred.

Concierge will review the Workspace configuration and remove any duplicate or conflicting connections.

# FAQ

## Can I Fix This by Running Sync Again?

No. This error requires backend review to resolve the duplicate connection conflict.

## Does This Mean My Data Is Lost?

No. It means the Workspace has multiple QuickBooks Desktop connections configured.

## Can Multiple Users Connect QuickBooks Desktop?

Only one QuickBooks Desktop connection can be designated as the preferred exporter per Workspace.
