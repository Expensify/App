# How to troubleshoot common QuickBooks Desktop errors

When your QuickBooks Desktop integration encounters an issue, Expensify displays an error code in your report comments or in a notification to the Preferred Exporter. Use this page to find your error code and follow the link to the detailed troubleshooting steps.

---

- **DESK06**: Retrieve QuickBooks Web Connector logs when a record does not exist in QuickBooks Desktop. [Learn how to fix the DESK06 error](Export-Errors/DESK06-Export-Error.md).

- **DESK61**: Restore or activate missing classes in QuickBooks Desktop before exporting. [Learn how to fix the DESK61 error](Export-Errors/DESK61-Export-Error.md).

# FAQ

## Where do I find my QuickBooks Desktop error code?

When an export or sync fails, the error code appears in the report comments and in an email notification sent to the Preferred Exporter configured in your Workspace's **Accounting** settings.

## What should I do if my error code is not listed here?

If your error code does not appear on this page, try syncing your Workspace with QuickBooks Desktop first. If the issue persists, contact Concierge for assistance and include the full error message.
