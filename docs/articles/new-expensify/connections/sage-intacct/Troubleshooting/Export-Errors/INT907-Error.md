# INT907 Error: Invalid Connection Configuration

If you see the error message:

**“INT907 Error: Invalid connection configuration.”**

It means the Sage Intacct integration configuration is invalid or no longer properly authenticated.

This error indicates a connection-level issue rather than a problem with report data.

---

## Why the INT907 Error Happens

The INT907 error occurs when:

- The Sage Intacct connection settings are incomplete or corrupted  
- Authentication details are no longer valid  
- Required configuration steps were not completed  

Because this error relates to the integration configuration, it cannot be resolved by updating individual reports.

---

# How to Fix the INT907 Error

This error requires additional investigation.

Please reach out to **Concierge** for support. Include:

- The full error message  
- The Workspace name  
- The time the error occurred  

Concierge will review the connection configuration and help restore the integration.

---

# FAQ

## Can I fix this by running Sync?

No. This error indicates a configuration issue that typically requires backend review.

## Does this mean my integration is disconnected?

Not necessarily, but the configuration is invalid. Concierge can confirm whether the connection needs to be refreshed or reconfigured.
