# How to troubleshoot common NetSuite errors

When your NetSuite integration encounters an issue, Expensify displays an error code in your report comments or in a notification to the Preferred Exporter. Use this page to find your error code and follow the link to the detailed troubleshooting steps.

---

# FAQ

## Where do I find my NetSuite error code?

When an export or sync fails, the error code appears in the report comments and in an email notification sent to the Preferred Exporter configured in your Workspace's **Accounting** settings.

## What should I do if my error code is not listed here?

If your error code does not appear on this page, try syncing your Workspace with NetSuite first. If the issue persists, contact Concierge for assistance and include the full error message.
