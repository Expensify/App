# INT907 Error in Sage Intacct Integration

If you see the error:

INT907: Invalid connection configuration.

This means there is a configuration issue with the Sage Intacct integration, preventing syncs and exports from completing.

---

## Why the INT907 Error Happens in Sage Intacct

The INT907 error typically indicates:

- The Sage Intacct connection was not fully configured.
- Required setup fields were entered incorrectly.
- Sage Intacct credentials or configuration settings were changed.
- The integration configuration became invalid.

Because this error relates to the overall connection setup, all Sage Intacct exports and syncs may fail until it is resolved.

This is a connection configuration issue, not a journal or user permission error.

---

## How to Fix the INT907 Error

This issue requires review by Concierge.

1. Open **Concierge** in Expensify.
2. Reference the **INT907 Error**.
3. Provide the Workspace name.
4. Include any recent changes made to the Sage Intacct connection.

Concierge will review the Sage Intacct integration configuration and guide you through correcting the setup.

Do not disconnect the integration unless instructed, as this may clear configuration settings.

---

# FAQ

## Can I Retry the Sync?

You may retry the sync, but if the configuration is invalid, the error will likely persist until corrected.

## Does INT907 Affect All Exports?

Yes. If the connection configuration is invalid, all Sage Intacct exports and syncs may fail.

## Should I Disconnect and Reconnect the Integration?

Do not disconnect unless directed by Concierge.
