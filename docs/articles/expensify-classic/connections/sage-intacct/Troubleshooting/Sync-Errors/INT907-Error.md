# INT907 Error in Sage Intacct Integration

If you see the error:

INT907 Error: Invalid connection configuration.

This means there is a configuration issue with the Sage Intacct integration in your Workspace.

The connection settings between Expensify and Sage Intacct are incomplete, incorrect, or no longer valid.

---

## Why the INT907 Error Happens in Sage Intacct

The INT907 error typically indicates:

- The Sage Intacct connection was not fully configured.
- Required fields during setup were entered incorrectly.
- Sage Intacct credentials or configuration settings were changed.
- The integration configuration is no longer valid.

Because this error relates to the overall connection configuration, exports and syncs cannot proceed until it is resolved.

This is a connection configuration issue, not a Sage Intacct user permission or journal configuration error.

---

## How to Fix the INT907 Error

This error requires a review of the Sage Intacct integration settings.

Reach out to **Concierge** in Expensify and include:

- The full error message.
- Your Workspace name.
- Any relevant report IDs.
- The approximate time the error occurred.

Concierge will review the Sage Intacct connection configuration and guide you through correcting the setup.

---

# FAQ

## Should I Disconnect and Reconnect Sage Intacct?

Do not disconnect the integration unless directed by Concierge. Disconnecting may clear configuration settings and require a full reconfiguration.

## Does This Error Affect All Exports?

Yes. If the connection configuration is invalid, all Sage Intacct exports and syncs may fail until the issue is resolved.
