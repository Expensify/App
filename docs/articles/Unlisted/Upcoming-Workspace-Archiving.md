# Workspace Archiving (Replacing Deletion) (Upcoming — New Expensify)

**Status:** Planned / in development — subject to change; not yet generally available.

Instead of permanently deleting a workspace, admins will be able to archive it — preserving history and settings while removing it from active use. Archived workspaces can be restored if needed.
